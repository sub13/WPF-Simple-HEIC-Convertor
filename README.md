## Possibilities
* Dark and Light theme; 
* Preview;
* You can convert your HEIC images without constrains;
* Cute Design;

## Stack of technologies
* C#; 
* WPF;
* .NET FRAMEWORK;
* Material design pack;

## System requirments
* Windows 7-10; 
* Net Framework version not bellow 4.7;
